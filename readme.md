# Sopranos QuoteR
Maintainer: Rebecca Hadi 

Get quotes from the Sopranos by family member. 

Load as R package ('sopranos')

Requirements: 

* tidyverse 

* rvest 

Quotes scraped from: https://www.thedad.com/the-sopranos-quotes/
